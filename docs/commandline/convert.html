convert

Converts this Image from the Boxfuse Vault into an AWS AMI..

Usage: boxfuse convert image

> boxfuse convert hello:1.0

Waiting for AWS to create an AMI for axelfontaine/hello:1.0 in eu-central-1 (this may take up to 50 seconds) ...
AMI created in 00:12.603s -> ami-74d4d369

Settings

Argument boxfuse.conf property Default Description
-env=... env dev Use the AWS region of this Boxfuse environment to create the AMI
-m false Make the output of the command machine-readable (JSON) instead of plain text

Machine-readable output (sample)

{
  "success" : "true",
  "result" : {
    "aws" : {
      "ami" : "ami-12345678",
      "region" : "us-east-1"
    }
  }
}
